# Dict

```python

# dict / hashmap
# a dict is represented by a sequence of definitions
a:1 b:2 #= a:1 b:2
m1 =
  a:1
  b:2
m1 = a:1 b:2

s1 = "apple"
s2 = "banana"
fruit =
  (s1):4 # use variable values by wrapping in parens
  (s2):2
  orange:3 # default is to use string representation for definition
fruit.apple #= 4
fruit.orange #= 3

```
